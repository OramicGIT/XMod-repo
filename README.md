# XMod

## About

XMod is an physics sandbox game where you can build ANYTHING. 

## Getting Started

1. Download the latest release from [itch.io](https://oramix.itch.io/xmod)
2. Extract the archive
3. Run the executable (.exe or .x86_64)
NOTE: To run this game on Linux, go to your game folder in the terminal, then type "chmod +x XMod.x86_64". This allows to run the game executable.
NOTE: Sorry, no MacOS version.

## Modding

XMod supports community mods. Browse and download mods on [ModDB](https://moddb.com/games/xmod).
Place them in here:
Windows: C:\Users\%USER%\AppData\LocalLow\OramiX\XMod\Mods
Linux: /home/<user>/.config/unity3d/OramiX/XMod/Mods
NOTE: These folders are hidden. To show them, press Ctrl + H on Linux or the Folder Settings > Show hidden folder (set to True)

## Building from source code 

### Requirements

- Unity 2022.3.62f3 (not older and not newer)
- Packages: NetCode for GameObjects, Authentication, Relay (alredy installed in the src/ folder)

### Steps

1. Clone the repository:
   ```
   git clone https://github.com/OramicGIT/XMod-repo.git
   ```
1.1. Unpack the Assets.zip in the src/ folder
2. Open the project in Unity 2022.3.61f (or any other version higher than 2022)
3. Open the main scene (Menu) and hit Play to test, or build via **File → Build Settings**
